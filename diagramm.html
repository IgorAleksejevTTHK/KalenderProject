2Gantt-diagrammi kohandamine

Gantt Chart on MS Projecti põhivaade, mis kujutab ülesandeid horisontaalsete ribadena ajaloonel. Diagrammi saab kohandada Gantt Chart Wizard'i abil.

  1. Veendu, et oled Gantt Chart vaates (View → Gantt Chart).
  2. Klõpsa menüüribal Format.
  3. Vali Gantt Chart Style – siit saad muuta ribade värve ja stiili.
  4. Klõpsa Format → Bar Styles, et määrata erinevate ülesandetüüpide (summary, milestone, normal) väljanägemine.
  5. Teksti kuvamiseks ribade kõrval vali Format → Bar ja seadista Left/Right/Top/Bottom/Inside tekstiväljad.
🖼️ Pildilõige: Gantt-diagramm koos kohandatud ribavärvidega (Lisa siia oma ekraanipilt)
💡 Nõuanne: Kriitilise tee kuvamiseks mine Format → Critical Tasks ja luba Show Critical Path. Kriitilised ülesanded kuvatakse punaselt.

3Network Diagram – sõltuvuste kaart

Network Diagram (tuntud ka kui PERT-diagramm) kuvab ülesandeid kastidena, mis on ühendatud nooltega vastavalt sõltuvustele. See aitab mõista projekti loogilist ülesehitust.

  1. Mine View → Network Diagram.
  2. Iga ülesanne kuvatakse kastina, mis sisaldab nime, ID-d, kestust ja vastutajat.
  3. Kastide paigutuse muutmiseks mine Format → Layout.
  4. Kasti sisu kohandamiseks paremklõpsa kastil → Box Styles.
  5. Uute sõltuvuste loomiseks lohista ühe kasti servast teisele.
🖼️ Pildilõige: Network Diagram vaade ülesannete sõltuvustega (Lisa siia oma ekraanipilt)
🔷 Kriitilise tee tuvastamine: Tumedamate servadega kastid Network Diagram vaates tähistavad kriitilise tee ülesandeid – need on ülesanded, mille hilinemine lükkab edasi kogu projekti lõppkuupäeva.

4Ressursikasutuse graafik

Resource Graph näitab visuaalselt, kuidas ressursid on ajas jaotunud. See aitab tuvastada ülekoormuse ja alakasutuse perioode.

  1. Mine View → Resource Graph.
  2. Ülekoormatud ressursid kuvatakse punaste tulpadega.
  3. Kasuta all-vasakul olevaid nooli ressursside vahel liikumiseks.
  4. Muuda kuvataavat ajaühikut: View → Timescale → vali päevad, nädalad jne.
  5. Ülekoormuse lahendamiseks kasuta Resource → Level All automaatseks tasandamiseks.
🖼️ Pildilõige: Resource Graph – tulpdiagramm ressursikoormusega (Lisa siia oma ekraanipilt)
💡 Nõuanne: Ressursi vaates saab kuvada tunde, tööühikuid, kulusid või ülejäänud töömahtu – vaheta Format → Details alt soovitud näitaja.

5Vaate eksportimine ja jagamine

Valmis diagramme ja vaateid saab eksportida teistele jagamiseks mitmel viisil:

  1. Kopeeri pildina: Vali Task → Copy → Copy Picture, et kopeerida aktiivne vaade lõikelauale pildina.
  2. Prindi: File → Print – saad valida prinditava vaate ja ajaulatuse.
  3. Ekspordi PDF: File → Export → Create PDF/XPS Document.
  4. Salvesta pildina: Kleebi kopeeritud pilt Paint, Word või PowerPoint programmi ja salvesta.
🖼️ Pildilõige: Copy Picture dialoogaken (Lisa siia oma ekraanipilt)
📤 Jagamisnõuanne: Kasuta Copy Picture funktsiooni, kui soovid diagrammi lisada esitlusse – see loob täpse ekraanipildi hetkel kuvatavast vaatest koos kõigi kohanduste ja värvistustega.
