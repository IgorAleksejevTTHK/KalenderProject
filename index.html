MS Project kalendri kasutamine

1. Kuidas luua uus kalender

Uue kalendri loomiseks ava Microsoft Project ja vali:
Project → Change Working Time → Create New Calendar.
Seejärel sisesta kalendri nimi ja vali, kas kopeerida olemasolev kalender või luua tühi.

2. Kuidas muuta tööaegu

Tööaegade muutmiseks ava:
Project → Change Working Time.
Seal saad muuta tööpäevi, lisada puhkepäevi ja määrata tööajad.

  • Tööpäevade muutmine (nt esmaspäev–reede)
  • Tööaegade määramine (nt 9:00–17:00)
  • Eripäevade lisamine (puhkus, pühad)

3. Kuidas kalendrit projektis kasutada

Kalendri kasutamiseks projektis mine:
Project → Project Information
ja vali loodud kalender.

4. Kuidas kalender mõjutab ajakava

Kalender määrab, millal töö saab toimuda. Kui päev on märgitud puhkepäevaks, siis ülesanded ei liigu sellel päeval edasi. See mõjutab projekti tähtaegu ja kestust.

Näiteks kui lisada puhkusenädal, siis kõik ülesanded nihkuvad edasi.
