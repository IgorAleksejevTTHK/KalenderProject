# 📊 KalenderProject – ProjectLibre

Projekt on loodud eesmärgiga õppida projektihaldust kasutades **ProjectLibre** tarkvara.

---

## 📌 Projekti ülevaade

Selles harus on koostatud projektiplaan, mis sisaldab:

- ülesannete struktuuri (WBS)
- ajakava (Gantt diagramm)
- ülesannete sõltuvusi
- projekti kalendri kasutamist

ProjectLibre on tasuta alternatiiv Microsoft Projectile, mis võimaldab hallata projekte professionaalsel tasemel.

---

## 🎯 Projekti eesmärk

- õppida projektihaldust
- koostada täielik projektiplaan
- kasutada kalendreid ja tööaegu
- mõista ülesannete seoseid
